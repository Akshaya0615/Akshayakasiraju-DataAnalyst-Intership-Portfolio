# Data Analyst Internship Portfolio

## About

This repository showcases the projects completed during my Data Analyst Internship at ApexPlanet Software Pvt. Ltd. The internship provided hands-on experience in data cleaning, SQL analysis, data visualization, data storytelling, statistical validation,GitHub Portfolio.

---

## Internship Tasks

### Task 1 – Data Cleaning
- Cleaned and preprocessed raw datasets.
- Handled missing values and duplicates.
- Standardized data for accurate analysis.

### Task 2 – SQL Analysis
- Performed SQL queries to answer business questions.
- Used filtering, grouping, sorting, and aggregate functions.
- Extracted meaningful business insights.

### Task 3 – Power BI Dashboard
- Developed an interactive Power BI dashboard.
- Created KPI cards and visualizations.
- Presented key business insights using charts.

### Task 4 – Data Storytelling & Statistical Validation
- Converted analytical findings into a business story.
- Performed hypothesis testing to validate insights.
- Created a presentation summarizing key findings.

### Task 5 – Capstone Integration & Portfolio Finalization
- Created a professional GitHub portfolio.
- Documented all internship tasks.
- Compiled project links, presentation, and key learnings.
